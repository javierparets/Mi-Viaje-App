# Mi Viaje - Planificador de Vacaciones en Familia

Una PWA (Progressive Web App) completa para planificar vacaciones en familia.

## 🚀 Estructura del Proyecto

```
mi-viaje/
├── src/
│   ├── components/
│   │   └── MiViaje.jsx
│   ├── App.jsx
│   ├── main.jsx
│   └── index.css
├── index.html
├── manifest.json
├── package.json
├── vite.config.js
├── tailwind.config.js
├── postcss.config.js
├── netlify.toml
└── .gitignore
```

## ✨ Características

- 🌍 Agregar países con fechas
- 💰 Trackear gastos por país
- 📍 Lugares de interés con horarios y precios
- 📅 Itinerario diario
- 🎒 Lista de packing
- 📄 Guardar documentos importantes
- 📸 Galería de fotos
- 💾 Almacenamiento local (offline)

## 📱 Instalación Local

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Compilar para producción
npm run build
```

## 🌐 Deployment en Netlify

1. Sube este proyecto a GitHub
2. Ve a [netlify.com](https://netlify.com)
3. Click en "New site from Git"
4. Selecciona tu repositorio
5. Netlify configurará todo automáticamente (lee el netlify.toml)
6. ¡Listo! Tu app estará en vivo en segundos

## 💡 Uso

- **Dashboard** - Resumen y presupuesto
- **Países** - Agrega países y su información
- **Itinerario** - Planifica actividades
- **Packing** - Qué llevar
- **Docs** - Pasaportes y reservas
- **Fotos** - Memories del viaje
